# HERBIE - Humanoid Experimental Robot Bio-type Integrated with Electronic

## Overview
HERBIE is a healthcare companion humanoid robot developed by me as a BCA project. The system combines robotics, embedded systems, bio-sensor monitoring and AI-assisted interaction into a single integrated platform.

The robot is capable of monitoring body temperature, heart rate and blood oxygen levels using DHT11 and MAX30100 sensors. HERBIE operates on a wheeled robotic platform and supports multiple control methods including Bluetooth mobile application control, voice commands and IR remote control.

The project uses Arduino Uno and Arduino Mega for hardware control, while a mini Windows 11 computer manages the Python-based voice assistant and speech recognition system. A 7-inch display is used for visual monitoring and interaction.

## Features
